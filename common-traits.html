Debug

Sometimes, especially when troubleshooting, we want to quickly inspect the contents of a variable, such as the origin instance of Point below.

struct Point {
    x: i32,
    y: i32,
}

fn main() {
    let origin = Point { x: 0, y: 0 };
    println!("{:?}", origin); // not work
}

The Debug trait enables this by allowing types to be printed using the {:?} formatter in macros like println!.
